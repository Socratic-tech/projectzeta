Every school needs their own copy of the data spreadsheet.

Create Your Spreadsheet

Clicking the button will prompt you to make your own copy.

1. Install Backend Code

2. Open your Google Sheet → Extensions → Apps Script.

3. Paste the backend script into Code.gs → Save.

4. Run initialSetup() to create system tabs.

5. Deploy Web App → New Version → Anyone → Copy URL.

6. Paste that URL below to connect your database.
