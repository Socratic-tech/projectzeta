2 Deploy to Google

  1. Open a new Google Sheet.
  2. Go to Extensions > Apps Script.
  3. Paste the code you copied above.
  4. CRITICAL: Select 'initialSetup' from the toolbar dropdown and click 'Run'. Accept permissions.
  5. Click Deploy > New Deployment.
  6. Select type: Web App.
  7. Execute as: Me.
  8. Who has access: Anyone.
  9. Click Deploy and copy the Web App URL.

Troubleshooting

  • Ensure you clicked "Run" on initialSetup first.
  • Ensure access is set to "Anyone" (not "Anyone with Google Account").
  • If you change the code later, you must do a New Deployment.
