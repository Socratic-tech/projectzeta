2 Google Sheet Setup

  1. Open a new Google Sheet.
  2. Go to Extensions > Apps Script.
  3. Important: Delete any existing text in the code editor first!
  4. Paste the code you copied from Step 1.
  5. Click the "Save" icon (Floppy Disk) in the toolbar.
  6. In the toolbar dropdown, select initialSetup and click Run.
  7. Accept permissions (Review Permissions > Choose Account > Advanced > Go to (Unsafe) > Allow).
  8. Note: This creates the "Settings" sheet with your default password.

3 Deploy

  1. Click Deploy > New Deployment (Top Right).
  2. Click the Gear Icon > Select Web App.
  3. Description: "V1".
  4. Execute as: Me.
  5. Who has access: Anyone (Vital for the app to work).
  6. Click Deploy. (You may be asked to authorize a second time).
  7. Copy the Web App URL generated.
